# KakaoBotSsafy
싸피 코치의 출퇴근 및 업무를 위한 일정관리 봇   
프론트엔드 없이 파싱을 이용해 채팅 해주는 봇임.

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker
<!-- 도커 사용법 -->
```bash
# 도커 이미지 빌드
$ docker build -t kakao-bot-ssafy .

# 도커 컨테이너 실행
$ docker run -d -p 3000:3000 --name kakao-bot-ssafy kakao-bot-ssafy
```
